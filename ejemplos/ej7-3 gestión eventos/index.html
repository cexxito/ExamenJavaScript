Anular eventos: removeEventListener

Los eventos se capturan de forma indefinida; sin embargo, en algunas ocasiones nos puede interesar que un evento se capture solo una vez. Para estos casos, se puede usar el método remoceEventListener. El método anulará la función asociada a un evento, pero esta no puede ser anónima.

Capturar elementos dinámicos

Cuando se crea un "escuchador" de eventos, este se asocia a los elementos indicados siempre que existan en el DOM. Lógicamente, si un elemento se crea después, no se capturará ningún evento asociado a él si no se indica después de su creación.
