Input

Hay cuatro elementos de tipo entrada de texto: text, password, textarea y hidden. Para poder usar estos objetos, podemos acceder a ellos a través de los campos id o name que asignemos a cada uno. Lo recomendable es que ambos campos se llamen igual.

Cuando se envían los datos de un formulario al servidor, lo que se envía son los atributos name y sus valores (atributo value). El valor siempre será una cadena de texto.
